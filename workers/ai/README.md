# SellerCanvas AI Worker

这个目录用于承载未来独立运行的 AI 任务执行器。

Worker 的职责是执行耗时任务，不直接面向客户页面：

- 图片分析任务
- Prompt 生成任务
- 批量生图任务
- 视觉质量检查任务
- Listing 文案生成任务
- 导出打包任务
- 任务失败后的积分退回协调

标准流程：

1. 客户在前端点击功能按钮。
2. 后端 API 创建任务并冻结对应积分。
3. Worker 拉取或接收任务。
4. Worker 调用 OpenAI、图片模型或其他 Provider。
5. 成功时写入结果并确认扣除积分。
6. 失败时标记失败并退回冻结积分。

客户不会接触 Worker，也不会看到开发者 API Key。
