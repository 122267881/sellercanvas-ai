# SellerCanvas 客户端

这个目录用于承载未来正式上线的客户使用网站。

客户站只面向付费用户，负责完整的使用流程：

- 用户注册、登录和第三方 OAuth 登录
- 查看订阅方案、购买套餐和管理账单
- 查看当前可用积分
- 创建商品项目
- 上传商品图片
- 进行 AI 商品分析
- 反推并生成导演级 Prompt
- 批量生成商品图
- 生成和编辑 Listing 文案
- 导出 Listing Pack
- 查看历史记录和个人数据

客户站不能出现这些开发者功能：

- API Key 创建和配置
- AI Provider 密钥配置
- 客户列表
- 全局用量统计
- 审计日志
- 系统健康和内部配置

迁移期间，根目录的 `index.html` 和 `app.js` 仍然作为可运行的旧客户站保留。新客户站开发完成并通过验收后，再逐步替换旧页面。
