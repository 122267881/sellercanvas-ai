# SellerCanvas 开发者管理后台

这个目录用于承载未来正式上线的开发者/运营管理后台。

管理后台只给开发者和运营人员使用，不能开放给普通客户。

后台负责：

- 管理员登录和权限校验
- 客户列表、搜索、筛选和导出
- 客户订阅状态查看
- 支付记录和发票记录
- 积分账户和积分流水
- AI 任务监控
- AI Provider 配置
- 开发者 API Key 管理
- API 调用用量分析
- 审计日志
- CSV 数据导出

权限要求：

- 客户账号永远不能访问管理后台页面
- 客户前端不能看到 API Provider 配置
- 开发者 API Key 只能在后台创建、停用和查看调用数据

迁移期间，根目录的 `admin.html` 和 `admin.js` 仍然作为可运行的旧后台原型保留。新后台稳定后，再切换到独立后台站点。
